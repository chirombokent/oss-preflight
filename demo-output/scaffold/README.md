# Discord Summary Bot

A Discord bot that summarizes channel activity, built with discord.js.

## Installation

```bash
npm install
```

## Configuration

Create a `.env` file with your Discord bot token:

```
DISCORD_TOKEN=your_token_here
```

## Running

```bash
npm start
```

## Testing

Run the smoke test (uses mocked messages, no credentials required):

```bash
npm test
```

- **Package**: discord.js@14.26.4
- **Repository**: https://github.com/discordjs/discord.js

## Next Steps

1. Get a Discord bot token from https://discord.com/developers/applications
2. Add the token to your `.env` file
3. Invite the bot to your server
4. Run `npm start` to start the bot
