> The single summary of every meaningful Bob task. Each relevant session also
> has a `bob_sessions/S<id>-<slug>/` folder with exported task-history markdown
> and, where available, a task-session consumption-summary screenshot.

## Export Notes

- **Export UI path:** Bob IDE History -> select task -> task header -> save task-session consumption summary screenshot -> Export task history.
- **Task-history file format:** Markdown task-history exports are stored in each `bob_sessions/S<id>-<slug>/` folder where available.
- **Screenshot saved as:** `consumption-summary.png`, `consumption_summary.png`, or the exported session screenshot filename in the session folder.
- **Markdown saved as:** `task-history.md` or `task_history.md` in the session folder.
- **`skill` group on `reviewer` accepted by this Bob build?** Yes in `.bob/custom_modes.yaml`; Advanced remains the guaranteed runtime for skills.
- **Checkpoints work?** Not re-verified during P7 continuation; git commits and exported histories are the safety net.
- **Bob commit-message sparkle works?** Not re-verified during P7 continuation; final commit requires human approval first.
- **Bob Shell available?** Not required for the submission path.
- **BobCoin usage note:** IBM clarified provisioned BobCoins are free to use with the hackathon accounts; record meaningful session usage and measurable outcomes rather than treating 40 as a hard charge cap.
